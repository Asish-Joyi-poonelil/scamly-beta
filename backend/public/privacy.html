Privacy

Scamly Beta Privacy Policy

What Scamly accesses

Scamly reads the visible email content shown on supported Gmail, Outlook, and Yahoo Mail pages so it can detect likely scam and phishing signals for the user.

What stays local

  • The local first-pass scan runs in the browser.
  • The extension stores only the latest scan summary locally so the popup can show the user’s recent result.

Optional Deep AI Check

Deep AI Check is off by default. If a user turns it on, Scamly sends the visible email text, sender text, subject, and visible links from the opened email to the user-configured Scamly backend so the backend can request an AI analysis.

The backend should keep the OpenAI API key on the server only. The extension must not expose the API key in client-side code.

What Scamly does not do

  • Scamly does not sell personal data.
  • Scamly does not use accessed email data for advertising.
  • Scamly does not read unrelated websites outside its supported webmail pages.

Feedback

If a user submits feedback, Scamly stores only the information the user submits through the feedback form or fallback contact method.
